    Intro To JSON - Javascript Object Notations
  1. Used to Store and exchange data
  2. We can use languages like JSON, XML(Extensible Markup Language) to exchage data between server and clients (user)
    3. Advantages of JSON over XML
    • Text Based Format
    • Light Weight
    • Human Readable
    • Language Independent
    • Support all popular Programming langu eg. PHP, JAVA, Python
    • Easy to Organise and access
    4. DisAdvantages of JSON
    • Does not support comments and namespaces
    • Can not use it for transfer video, audio, imagesor any other binary info
    5. Advantages of XML over JSON
    • Supports Comments and Namespaces
  6. Difference between Javascript Object and JSON

					Writing Method 3 (Writing keys with space, single or double quotation mark is imp this time)
					let Person = { "first Name": "Dinesh", "last Name": "Jadhav"};
					Printing in Javascript Objects and JSON is same
					alert(Person.firstName .+""+. Person.lastName);
				
    					Only one method of writing (We can not use single quotation marks also while writing key)
